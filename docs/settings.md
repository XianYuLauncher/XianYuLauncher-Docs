# 启动器设置

## 打开设置

1. 在启动器主界面，点击侧边栏的 "设置" 图标
2. 进入设置界面
3. 可以看到各种设置选项

## 基础设置

### 个性化设置

1. 在设置界面，点击 "个性化" 选项
2. 在个性化设置中，可以：
   - 切换深/浅主题
   - 切换Mica/MicaAlt/Acrylic/自定义背景主题

### 语言设置

1. 在设置界面，点击 "个性化" 选项
2. 在语言设置中，选择您想要使用的语言
3. 支持的语言包括：
   - 中文（简体）
   - English
   - ...
4. 启动器重启后会自动切换到所选语言

## 下载设置

### 下载源

1. 在设置界面，点击 "网络与下载" 选项
2. 在下载源设置中，选择您想要使用的下载源
3. 支持的下载源包括：
   - 官方源
   - [BMCLAPI](https://bmclapi2.bangbang93.com/)
   - [MCIM](https://www.mcimirror.top/)

### 自动下载前置Mod

1. 在设置界面，点击 "网络与下载" 选项
2. 在自动下载前置Mod设置中，可以：
   - 启用/禁用自动下载前置Mod

## 游戏设置

### Java 设置

1. 在设置界面，点击 "Java设置" 选项
2. 在 Java 设置中，可以：
   - 自动查找 Java 路径
   - 手动指定 Java 路径
