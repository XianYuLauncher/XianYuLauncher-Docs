# 多人游戏

## 加入服务器

### 直接加入服务器

1. 启动 Minecraft，进入主界面
2. 点击 "多人游戏"
3. 点击 "直接连接"
4. 输入服务器地址
5. 点击 "连接" 按钮
6. 等待连接到服务器

### 添加服务器到列表

1. 启动 Minecraft，进入主界面
2. 点击 "多人游戏"
3. 点击 "添加服务器"
4. 输入服务器名称
5. 输入服务器地址
6. 点击 "完成" 按钮
7. 服务器会添加到服务器列表中
8. 点击服务器名称即可连接

## 陶瓦联机服务

::: info
XianYuLauncher 的联机功能由 **陶瓦联机** 服务提供支持。
:::

XianYuLauncher 内置了联机服务，让您可以轻松与朋友一起游戏：

1. 在启动器主界面，点击侧边栏的 "联机" 图标
2. 可以：
   - 创建房间
   - 加入朋友的房间
   - 查看房间成员
4. 在启动页，点击 "启动游戏" 按钮
5. 进入游戏后点击 "多人游戏" ，会出现一个醒目的房间区域,点击 "加入房间" 即可加入房间

- 你的网卡了。

### 连接超时

- 你的网卡了。
