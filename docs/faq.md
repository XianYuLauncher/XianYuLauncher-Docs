# 常见问题

## 启动器问题

### 启动器无法启动

**问题**：双击启动器快捷方式后，启动器没有反应。

**解决方法**：
- 检查系统是否满足最低要求（Windows 10/11 X64/X86/ARM64）
- 检查是否安装了.NET 10.0 运行时与WinAppSDK 1.8
- 检查启动器安装目录的权限
- 重新安装启动器或加群:1002593659 @咸鱼 进行反馈

### 启动器崩溃

**问题**：启动器在运行过程中崩溃。

**解决方法**：
- 检查启动器是否为最新版本
- 关闭其他占用大量内存的程序
- 检查启动器日志获取详细错误信息

## Minecraft 问题

### 游戏无法启动

**问题**：点击 "启动游戏" 按钮后，游戏没有反应或立即关闭。

**解决方法**：
- 检查 Java 安装是否正确
- 调整游戏内存分配
- 检查是否安装了不兼容的Mod
- 查看游戏日志获取详细错误信息

### 游戏卡顿

**问题**：游戏运行时出现卡顿或掉帧。

**解决方法**：
- 降低游戏画质设置
- 增加游戏内存分配
- 关闭后台运行的程序
- 删除一些不常用的Mod
- 优化游戏设置

### 游戏崩溃

**问题**：游戏在运行过程中崩溃。

**解决方法**：
- 检查是否安装了不兼容的Mod
- 检查游戏版本和加载器版本是否匹配
- 尝试更新游戏到最新版本
- 尝试更新加载器到最新版本
- 查看崩溃报告获取详细信息

## 账号问题

### 无法登录账号

**问题**：输入账号信息后，无法登录 Minecraft 账号。

**解决方法**：
- 检查账号密码是否正确
- 检查网络连接是否正常
- 联系 Mojang 支持

## 网络问题

### 下载速度慢

**问题**：下载游戏版本或模组时速度很慢。

**解决方法**：
- 检查网络连接是否正常
- 关闭其他占用网络的程序
- 尝试更换网络环境
- 尝试在设置页更改下载源
- 尝试使用加速器

## 其他问题

### 存档丢失

**问题**：游戏存档丢失。

**解决方法**：
- 尝试搜索.Minecraft/您的版本(关闭版本隔离请忽略)/saves/文件夹

### 模组无法正常工作

**问题**：安装的Mod无法正常工作。

**解决方法**：
- 检查Mod是否与游戏版本兼容
- 检查Mod是否与其他Mod冲突
- 尝试在版本管理中更新Mod
- 检查Mod的依赖是否安装
- 查看Mod的说明文档
